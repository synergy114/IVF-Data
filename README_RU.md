# Перенос эмбрионов на 3-й vs 5-й день: анализ исходов ЭКО

Сравнительный анализ переноса эмбрионов на стадии дробления (День 3) и на стадии бластоцисты (День 5) с использованием клинических данных ЭКО. Все анализы выполнены в R.

---

## Данные

Набор данных включает **222 цикла ЭКО** от **177 уникальных пациенток**, часть из которых участвовала в нескольких циклах. Данные представлены в двух листах:

- **`cycles`** — одна строка на цикл; содержит демографические характеристики пациенток (возраст, ИМТ, гравидность, происхождение спермы), лабораторные показатели (количество ооцитов, оплодотворение, развитие эмбрионов) и клинические исходы (имплантация, беременность, живорождение)
- **`Oocyte-level embryo development`** — одна строка на ооцит; содержит стадию зрелости при пункции, статус оплодотворения и стадию бластоцисты на 5-й день

> Исходные файлы с данными в репозиторий не включены.

---

## Анализируемые показатели

**Лабораторные**
- Количество полученных ооцитов, ооцитов MII, доля MII (%)
- Количество оплодотворённых ооцитов (2PN), частота оплодотворения (%)
- Количество эмбрионов на 3-й день, частота дробления (%)
- Количество перенесённых эмбрионов
- Количество бластоцист на 5-й день, частота бластуляции (%) — *только циклы с переносом на День 5*

**Клинические**
- Имплантация (≥1 гестационный мешок)
- Частота имплантации (число мешков / число перенесённых эмбрионов)
- Клиническая беременность
- Живорождение (> 24 недель)

---

## Статистические методы

| Переменная | Единица анализа | Метод |
|---|---|---|
| Возраст, ИМТ | Пациентка | Критерий Вилкоксона |
| Возрастная группа, группа ИМТ, происхождение спермы | Пациентка | Критерий χ² |
| Непрерывные показатели цикла | Цикл | ОУУ — гауссовская |
| Бинарные исходы | Цикл | ОУУ — биномиальный логит |
| Частота имплантации | Цикл | ОУУ — гауссовская |

- **ОУУ** (обобщающие уравнения оценивания, GEE) — модели с обменной структурой корреляции, кластеризованные по пациентке, для учёта повторных циклов
- Значимость оценивается по **критерию Вальда ОУУ** (χ², 1 степень свободы)
- Описательная статистика представлена как **среднее ± СОС** и **медиана [МКИ]**
- Частоты выражены в процентах; частота имплантации ограничена 100%
- Показатели бластоцист представлены описательно только для циклов с переносом на День 5
